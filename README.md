# 🎟️ Event Management System

## 📘 Project Description
The **Event Management System** is a web-based application designed to help universities and student clubs organize and manage events easily.  
Students can browse upcoming events, register or cancel their participation, and receive reminders.  
Organizers can create and manage events, track participants, and collect feedback.  
The system aims to simplify event planning and improve student engagement using a simple and user-friendly interface.
